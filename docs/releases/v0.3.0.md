## go-ddgs-stealth v0.3.0

Stealth foundation release.

### Highlights

- Unified fetcher stack:
  - `HTTPFetcher` (anti-bot transport reuse)
  - `StealthyFetcher` (Rod-backed)
  - `DynamicFetcher` (interactive flow wrapper)
- Adaptive parsing with selector self-healing and persistence.
- Spider/crawler runtime with:
  - concurrency,
  - domain pacing,
  - checkpointing,
  - JSONL streaming,
  - pause/resume,
  - depth and host policy controls.
- New stealth HTTP endpoints:
  - `POST /v1/stealth/fetch`
  - `POST /v1/stealth/crawl`
- Stealth endpoint safety controls:
  - optional bearer auth via `GODDGS_API_TOKEN`
  - per-IP rate limiting via `GODDGS_STEALTH_RATE_PER_MIN`
- Observability extensions:
  - Prometheus stealth metrics
  - OpenTelemetry span hooks for stealth request lifecycle
- CLI extensions:
  - `stealth-fetch`
  - `stealth-crawl`
  - `repl`
- Release workflow now publishes checksums (`SHA256SUMS.txt`).

### Quality Gates

- `go test ./...`
- `go build ./...`
- coverage gate `>=85.0%`

### Notes

- `go-ddgs` and `go-ddgs-stealth` are separate projects and repos.
- Anti-bot behavior remains target-dependent and can degrade as upstream protections change.
